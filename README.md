# 마비노기 모바일 쿠폰 (Chrome 확장)

[마비노기 모바일 쿠폰 사이트](https://mcoupon.nexon.com/mabinogimobile/)에서 **회원코드(NPA_CODE)** 와 **쿠폰번호**를 빠르게 입력·등록하기 위한 Chrome 확장 프로그램입니다.

넥슨 로그인을 해도 사이트가 회원코드를 자동으로 가져오지 못하는 경우가 많아, 게임에서 확인한 회원코드를 확장에 저장해 두고 쿠폰 페이지에 자동으로 채워 넣을 수 있습니다.

## 주요 기능

- **다계정 관리** — 별명 + 회원코드를 여러 개 저장
- **기본 계정** — ★ 표시된 계정이 쿠폰 페이지에 자동 입력
- **계정 순서 변경** — ▲/▼ 버튼으로 목록 순서 조정
- **쿠폰 페이지 자동 열기** — 탭이 없으면 자동으로 열고 적용
- **API 직접 호출** — 사이트 버튼 클릭 대신 공식 API로 캐릭터 조회·쿠폰 등록
- **페이지 내 오류 표시** — 이미 사용한 쿠폰 등 오류를 확장 팝업과 쿠폰 페이지 양쪽에 표시
- **캐릭터 자동 선택** — 캐릭터가 1명일 때 자동 등록 (옵션)

## 설치 방법

1. Chrome에서 `chrome://extensions` 접속
2. 우측 상단 **개발자 모드** 활성화
3. **압축해제된 확장 프로그램을 로드합니다** 클릭
4. 이 프로젝트 폴더(`mabinogi_cupon`) 선택

> 확장을 업데이트한 뒤에는 반드시 **새로고침** 버튼을 눌러 주세요.

## 사용 방법

### 1. 회원코드 확인 (게임 내)

게임 접속 → 우측 상단 **≡** → **환경설정** → **계정** → **회원코드** 복사

### 2. 계정 등록 (확장 팝업)

1. 확장 아이콘 클릭
2. **새 계정** 또는 드롭다운의 **+ 새 계정 추가** 선택
3. **별명**과 **회원코드**(12~13자 영문/숫자) 입력
4. **추가** 클릭

### 3. 쿠폰 적용

1. 드롭다운에서 사용할 계정 선택
2. **쿠폰번호** 입력
3. (선택) **캐릭터 1명이면 자동 선택** 체크
4. **페이지에 적용 및 조회** 클릭

쿠폰 페이지가 열려 있지 않으면 자동으로 열리고, 회원코드·쿠폰번호가 입력된 뒤 캐릭터 조회가 진행됩니다.

### 계정 관리

| 버튼 | 설명 |
|------|------|
| **새 계정** | 입력란을 비우고 새 계정 추가 모드로 전환 |
| **추가 / 수정** | 새 계정이면 추가, 기존 계정 선택 시 수정 |
| **삭제** | 선택한 계정 삭제 |
| **▲ 위로 / ▼ 아래로** | 목록에서 계정 순서 변경 |
| **★ 기본** | 선택한 계정을 기본(자동 입력) 계정으로 설정 |

## 프로젝트 구조

```
mabinogi_cupon/
├── manifest.json      # 확장 설정
├── background.js      # 탭 열기·적용 요청 처리 (서비스 워커)
├── content.js         # 쿠폰 페이지 DOM/API 연동
├── popup.html         # 확장 팝업 UI
├── popup.js           # 계정·쿠폰 입력 로직
├── popup.css          # 팝업 스타일
└── icons/             # 확장 아이콘 (16, 48, 128)
```

## 권한 안내

| 권한 | 용도 |
|------|------|
| `storage` | 회원코드·계정 목록 저장 |
| `tabs` | 쿠폰 페이지 탭 찾기·열기·전환 |
| `https://mcoupon.nexon.com/*` | 쿠폰 사이트에서만 동작 |

## 동작 방식 (요약)

1. 팝업에서 **적용** 클릭 → `background.js`가 쿠폰 탭을 찾거나 새로 엽니다.
2. `content.js`가 페이지에 회원코드·쿠폰번호를 채웁니다.
3. 넥슨 쿠폰 API를 호출해 캐릭터를 조회합니다.
   - `POST /mabinogimobile/coupon/api/v1/characters-by-npacode`
   - `POST /mabinogimobile/coupon/api/v1/redeem-coupon-by-npacode`
4. 성공·실패 메시지를 쿠폰 페이지 알림 팝업과 입력란 아래에 표시합니다.

## 유의사항

- 회원코드는 **게임 설정 > 계정**에서 확인한 값과 정확히 일치해야 합니다. 공백 없이 입력하세요.
- 캐릭터가 여러 명이면 자동 선택 옵션을 켜도 **1명일 때만** 자동 등록됩니다. 그 외에는 페이지에서 직접 선택하세요.
- 이미 사용했거나 만료된 쿠폰은 사이트 정책상 등록할 수 없습니다.
- 이 확장은 넥슨 공식 프로그램이 아닙니다. 사용에 따른 책임은 사용자에게 있습니다.
